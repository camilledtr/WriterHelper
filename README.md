# Writer Helper FR

### Set up the app
In the **backend** folder :
`pip install Flask`
`pip install flask-cors`
(soon requirements.txt to be added)

In the **frontend** folder :
`npm i`

### Run the app

In the **backend** folder, run the server :
`python3 app.py`

In the **frontend** folder, run the client side :
`npm run dev`
